## Design Specification Notes

| Element | Specification |
|---|---|
| Emotion badge colors | Red-ish pill → Anger/Disgust; Gray → Neutral; Light green → Happy/Supportive |
| Stage connections | Downward arrows between the 4 colored boxes in right panel |
| Selected comment highlight | Dashed blue border + small annotation `"selected agent"` |
| Username anonymization | Format: `@User_XX_*YY` or `@Generic_Label`. No real-world identifiers. |
| Comment tone diversity | Must include: 1 angry, 1 neutral-analytical, 1 calm-rational, 1 mildly supportive |
| Font | Clean sans-serif (e.g., Inter, Helvetica Neue; or Source Han Sans for CJK fallback) |
| Hashtag bilingual | English label primary; Chinese version (`#预制菜透明化#`) optional as secondary smaller text |
| Figure dimensions | ~2400 × 1400 px, landscape orientation |
| Background | `#F5F5F5` light gray |
